   - Save as PNG for best quality

2. **Replace existing file:**
   ```bash

   mv public/og-image.png public/og-image-old.png
   
   # Add your new file
   # Save new file as: public/og-image.png
   ```

3. **Clear social media caches:**
   After deploying, clear cached images:
   
   **Facebook:**
   - Go to https://developers.facebook.com/tools/debug/

   - Click "Scrape Again"
   
   **Twitter:**
   - Go to https://cards-dev.twitter.com/validator

   - Click "Preview card"

   - Go to https://www.linkedin.com/post-inspector/

   Share your URL on:
   - Facebook (in a private post)
